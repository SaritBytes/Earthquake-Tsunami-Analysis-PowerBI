# Earthquake & Tsunami Occurrence Analysis (Power BI)

## Project Overview
This project provides a comprehensive analysis of global seismic activities. Using Power BI, I transformed raw geospatial data into an interactive dashboard to visualize disaster patterns and high-risk zones.

## Key Features
* **Interactive Dashboard:** Visualizes earthquake frequency and magnitude across the globe.
* **Tsunami Correlation:** Analyzes the relationship between seismic depth and tsunami alerts.
* **Geospatial Mapping:** Identifies high-risk clusters using Power BI map integrations.
* **Advanced DAX:** Built custom measures for trend analysis and historical comparisons.

## Technical Stack
* **Tool:** Power BI Desktop
* **Data Processing:** Power Query (ETL)
* **Calculations:** DAX (Data Analysis Expressions)
